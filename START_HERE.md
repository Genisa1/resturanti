### Understanding the Flow
1. User visits URL with `?page=xxx`
2. `index.php` router catches the request
3. Router loads appropriate Controller
4. Controller loads Model for data
5. Model queries Database
6. Controller loads View with data
7. View renders HTML to user

### Example: Viewing Products
```
URL: ?page=products
  ↓
index.php routes to ProductController
  ↓
ProductController loads Product model
  ↓
Product model queries database
  ↓
Controller passes data to products.php view
  ↓
View displays all products with HTML
```
